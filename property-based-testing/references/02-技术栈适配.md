# 技术栈适配

## Python / Hypothesis

### 安装

```bash
pip install hypothesis
```

### 基础用法

```python
from hypothesis import given, strategies as st

@given(st.integers(), st.integers())
def test_add(a, b):
    assert a + b == b + a
```

### 常用策略

```python
# 基础类型
st.integers(min_value=0, max_value=100)
st.floats(allow_nan=False)
st.text(min_size=1, max_size=50)
st.booleans()

# 集合
st.lists(st.integers(), min_size=1, max_size=10)
st.dictionaries(st.text(), st.integers())
st.sets(st.integers())

# Pydantic 模型（自动生成合法实例）
from pydantic import BaseModel
class User(BaseModel):
    name: str
    age: int

@given(st.from_type(User))
def test_user_model(user):
    assert isinstance(user.name, str)
    assert isinstance(user.age, int)

# 组合策略
st.one_of(st.integers(), st.text())           # 二选一
st.sampled_from(["GET", "POST", "PUT"])        # 从列表中选
st.tuples(st.integers(), st.text())            # 元组

# 自定义策略
from hypothesis.strategies import composite

@composite
def user_strategy(draw):
    name = draw(st.text(min_size=1, max_size=20))
    age = draw(st.integers(min_value=0, max_value=150))
    return User(name=name, age=age)

@given(user_strategy())
def test_custom_user(user):
    assert user.age >= 0
```

### 过滤输入

```python
from hypothesis import assume

@given(st.integers())
def test_positive_only(x):
    assume(x > 0)
    assert x > 0
```

### 回归用例

```python
@given(st.integers())
@example(0)       # 固定回归用例
@example(-1)
@example(999999)
def test_edge_cases(x):
    assert process(x) >= 0
```

### Stateful Testing（状态机）

```python
from hypothesis.stateful import RuleBasedStateMachine, rule

class APITester(RuleBasedStateMachine):
    def __init__(self):
        super().__init__()
        self.items = {}

    @rule(name=st.text(), price=st.floats(min_value=0))
    def create_item(self, name, price):
        response = client.post("/items", json={"name": name, "price": price})
        assert response.status_code == 201

    @rule()
    def list_items(self):
        response = client.get("/items")
        assert response.status_code == 200

TestAPI = APITester.TestCase
```

### 配置

```python
from hypothesis import settings

@settings(max_examples=200, deadline=500)  # 200次，每次500ms超时
@given(st.integers())
def test_slow(x):
    ...
```

## Dart/Flutter

### 工具选择

推荐 **glados**（简单轻量），备选 **kiri-check**（更活跃）。

### 安装

```yaml
# pubspec.yaml
dev_dependencies:
  glados: ^1.1.7
  test: ^1.24.0
```

### 基础用法

```dart
import 'package:glados/glados.dart';
import 'package:test/test.dart';

void main() {
  Glados<int>().test('addition identity', (a) {
    expect(a + 0, equals(a));
  });

  Glados2<int, int>().test('addition commutative', (a, b) {
    expect(a + b, equals(b + a));
  });
}
```

### 常用生成器

```dart
// 内置类型生成器
Glados<int>()              // 整数
Glados<String>()           // 字符串
Glados<List<int>>()        // 整数列表

// 自定义生成器
Glados(any.lowercaseLetter)
Glados(any.nonEmptyList(any.positiveIntOrZero))

// 多参数
Glados2<String, int>()
Glados3<int, int, int>()
```

### Flutter 数据类测试

```dart
// 测试 fromJson/toJson roundtrip
Glados2<String, int>().test('user json roundtrip', (name, age) {
  final user = User(name: name, age: age);
  final json = user.toJson();
  final restored = User.fromJson(json);
  expect(restored.name, equals(name));
  expect(restored.age, equals(age));
});
```

## C# / Unity（FsCheck）

### 安装

```bash
dotnet add package FsCheck.Xunit
```

### 基础用法（xUnit 集成）

```csharp
using FsCheck;
using FsCheck.Xunit;

public class MathTests
{
    [Property]
    public bool AdditionIsCommutative(int a, int b)
    {
        return a + b == b + a;
    }

    [Property]
    public bool SortPreservesLength(int[] list)
    {
        var sorted = list.OrderBy(x => x).ToArray();
        return sorted.Length == list.Length;
    }
}
```

### 自定义生成器

```csharp
// 自定义生成器
public class UserGenerators
{
    public static Arbitrary<User> UserGen()
    {
        var gen = from name in Arb.Generate<string>()
                  from age in Gen.Choose(0, 150)
                  select new User(name, age);
        return gen.ToArbitrary();
    }
}

[Property(Arbitrary = new[] { typeof(UserGenerators) })]
public bool UserRoundtrip(User user)
{
    var json = JsonSerializer.Serialize(user);
    var restored = JsonSerializer.Deserialize<User>(json);
    return user.Name == restored.Name && user.Age == restored.Age;
}
```

### 条件过滤

```csharp
[Property]
public Property DivisionByNonZero(int a, int b)
{
    // 跳过 b==0 的情况
    return (b != 0).When(
        () => a / b >= 0 || a / b < 0  // 不抛异常即可
    );
}
```

### Unity 注意事项

- 不要在 PBT 中直接测试 MonoBehaviour（生命周期由 Unity 控制）
- 提取纯逻辑到普通 C# 类，用 FsCheck 测试
- 物理计算逻辑可以提取为纯函数测试
